# AI-问答
一个AI-问答项目，接通GPTkey，用户可以登陆使用。用户输入问题得到回答，可以自行选择保存或清除记录。此外，用户可以浏览当日头条新闻，将新闻当作词条加入到提问内容中。

技术：Vue + MySQL + SpringBoot

项目仍在开发中。
